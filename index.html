Ice Cream Share ‚Äî Full-Stack Link Sharing App

Ice Cream Share is a full-stack web application I designed and developed for a university project. It provides a platform for ice cream enthusiasts to share, browse, and rate ice-cream-related links. The application combines a clean HTML/JavaScript front-end with a secure, RESTful API back-end powered by Node.js, Express, TypeScript, and PostgreSQL.

  • User registration and authentication (passwords securely hashed)
  • Post, rate, and hide links
  • Earn Cream Points through community ratings
  • RESTful API with no page reloads for core functions
  • Favourites list for positively rated links
  • Front page sorting (Most Recent, Highest Rated)
